Pipes included in Angular - Commonly in the Common Module

The Decimal Pipe performs numeric operations and formats the result. 

public totalSells: number = {{ totalSells }};
{{ (totalSells + 1235 ) | number:'1.2-2' }}

The Currency Pipe formats numbers as currency. 

public totalSells: number = {{ totalSells }};
{{ totalSells | currency:'HNL':'symbol-narrow':'1.4-4' }}

The Percent Pipe converts a number into a percentage. 

public percent: number = {{ percent }};
{{ percent | percent: '1.2-2' }}

The Exponential Pipe formats a number as exponential. 

public largeNumber: number = {{ largeNumber }};
{{ largeNumber | number: '1.2e-3' }}

The Fixed Pipe formats a number with a fixed number of digits after the decimal point. 

public pi: number = {{ pi }};
{{ pi | number: '1.2-2' }}
