En Angular, los pipes son transformadores de datos que permiten modificar la presentación de la información en la interfaz de usuario. A continuación, se presentan algunos pipes básicos comúnmente incluidos en el Common Module.

El Uppercase Pipe convierte el texto a mayúsculas. 

public nameLower: string = '{{ userName }}';
Resultado: {{ userName | uppercase }}

El Lowercase Pipe convierte el texto a minúsculas. 

public nameUpper: string = '{{ userName }}';
Resultado: {{ userName | lowercase }}

El Titlecase Pipe convierte la primera letra de cada palabra a mayúsculas. 

public fullName: string = '{{ userName }}';
Resultado: {{ userName | titlecase }}

También puedes utilizar el Date Pipe para formatear fechas personalizadas. 

public customDate: Date = new Date();
  • {{ customDate }}
  • {{ customDate | date }}
  • {{ customDate | date:'short' }}
  • {{ customDate | date:'long' }}
  • {{ customDate | date:'MMMM' }}
  • {{ customDate | date:'MMMM dd, yyyy' }}
  • {{ customDate | date:'long':'GMT-6' }}
  • {{ customDate | date:'long':'GMT-4' }}
  • {{ customDate | date:'long':'':'es-HN' }}
  • {{ customDate | date:'long':'':'en-US' }}
  • {{ customDate | date:'long':'':'fr-CA' }}
