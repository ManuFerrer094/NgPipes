In Angular, pipes are data transformers that allow modifying the presentation of information in the user interface. Below are some basic pipes commonly included in the Common Module.

The Uppercase Pipe converts text to uppercase. 

public nameLower: string = '{{ userName }}';
Result: {{ userName | uppercase }}

The Lowercase Pipe converts text to lowercase. 

public nameUpper: string = '{{ userName }}';
Result: {{ userName | lowercase }}

The Titlecase Pipe converts the first letter of each word to uppercase. 

public fullName: string = '{{ userName }}';
Result: {{ userName | titlecase }}

You can also use the Date Pipe to format custom dates. 

public customDate: Date = new Date();
  • {{ customDate }}
  • {{ customDate | date }}
  • {{ customDate | date:'short' }}
  • {{ customDate | date:'long' }}
  • {{ customDate | date:'MMMM' }}
  • {{ customDate | date:'MMMM dd, yyyy' }}
